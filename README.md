# Технології системного адміністрування — TSA-233

> **Курс:** Технології системного адміністрування
> **Аудиторія:** Курсанти 3 курсу ВІТІ
> **ОС:** Ubuntu 22.04 LTS (сервер), Windows 10/11 (клієнти)

Репозиторій містить методичні матеріали та конфігураційні файли для лабораторних і групових занять із адміністрування мережевих служб.

---

## Структура репозиторію

```
tsa233/
├── lesson6_2/          # DNS-сервер BIND9
│   ├── README.md
│   ├── configs/        # Конфігурація BIND
│   ├── docker/         # Docker-образ для стенду
│   └── docker-compose.yml
│
├── lesson6_4/          # Криптографія (лекція)
│   └── README.md
│
└── lesson7_3/          # Поштовий сервер Postfix + Dovecot
    ├── README.md
    └── configs/        # Конфігурація Postfix і Dovecot
```
